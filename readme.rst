# Codeigniter3 - Ecommerce

## Features

- Responsive ecommerce with the last version of Bootstrap
- CRUD users, items & more!
- Tables with great filters with DataTables
- Export items & users as Excel and/or PDF

## Tech
This ecommerce uses a number of open source projects to work properly:

- Bootstrap 5
- jQuery 3.6.0
- DataTables

## Installation
This ecommerce doesn't require any dependencies to run :)
